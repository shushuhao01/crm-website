## 🧭 智能部署引导

不确定选择哪种部署方式？回答以下几个问题，系统帮您推荐最适合的方案！

---

### 问题一：您的使用场景是什么？

| 场景 | 推荐方案 |
|------|---------|
| 只是想在本地电脑上试用体验 | 👉 **[点击查看：本地电脑部署教程 →](/docs/deploy/local)** |
| 要部署到服务器给团队正式使用 | 请继续看问题二 ⬇️ |

### 问题二：您是否有 Linux 运维经验？

| 您的情况 | 推荐方案 |
|---------|---------|
| 没有，我是服务器小白 | 🌟 **[点击查看：宝塔面板部署教程 →](/docs/deploy/bt)**（强烈推荐！有可视化界面） |
| 有一些基础，能敲 Linux 命令 | **[宝塔面板部署 →](/docs/deploy/bt)** 或 **[纯Linux部署 →](/docs/deploy/linux)** |
| 非常熟悉，有丰富运维经验 | 👉 **[点击查看：纯Linux部署教程 →](/docs/deploy/linux)** |

### 问题三：您是否已有云服务器？

| 您的情况 | 建议 |
|---------|------|
| 没有服务器 | 推荐购买 **[阿里云ECS →](https://www.aliyun.com/minisite/goods)** 或 **[腾讯云轻量 →](https://cloud.tencent.com/act/pro/lighthouse)**，新用户有优惠，2核2G 起步即可 |
| 有服务器，装了宝塔面板 | 👉 直接看 **[宝塔面板部署教程 →](/docs/deploy/bt)** |
| 有服务器，没装宝塔 | 👉 看 **[宝塔面板部署 →](/docs/deploy/bt)**（安装宝塔只需一条命令）或 **[纯Linux部署 →](/docs/deploy/linux)** |
| 只有 Windows/Mac 电脑 | 👉 看 **[本地电脑部署教程 →](/docs/deploy/local)** |

### 问题四：您会使用 Git 吗？

| 您的情况 | 获取代码方式 |
|---------|------------|
| 不会用 Git | 选择「下载压缩包」方式获取代码 |
| 会用 Git | 推荐「Git 克隆」方式，后续更新只需 `git pull`。公开仓库：**[github.com/shushuhao01/crm-system →](https://github.com/shushuhao01/crm-system)** |

### 问题五：是否需要 HTTPS？

| 您的情况 | 建议 |
|---------|------|
| 只是内部使用/测试 | HTTP 即可，暂时不需要 |
| 对外提供服务 | 强烈建议配置 HTTPS，各部署教程末尾都有 SSL 配置说明 |

---

## 📦 获取部署代码

购买私有部署后，您可以通过以下方式获取系统源代码：

**方式一：Git 克隆（推荐，方便后续更新）**

```bash
git clone https://github.com/shushuhao01/crm-system.git CRM
cd CRM
```

> 💡 后续版本更新只需执行 `cd CRM && git pull` 即可获取最新代码。

**方式二：下载压缩包**

前往 **[GitHub 仓库发布页 →](https://github.com/shushuhao01/crm-system/releases)** 下载最新版本压缩包，或在支付成功页面直接下载。

## 📊 三种部署方式对比

| 特性 | 本地电脑部署 | 宝塔面板部署 | 纯Linux部署 |
|------|-----------|-----------|-----------|
| 难度 | ⭐ 简单 | ⭐⭐ 中等 | ⭐⭐⭐ 较难 |
| 适合人群 | 个人体验/测试 | 小白用户/中小企业 | 运维工程师 |
| 操作系统 | Windows/macOS | Ubuntu/CentOS | Ubuntu/CentOS/Debian |
| 是否需要服务器 | ❌ 不需要 | ✅ 需要 | ✅ 需要 |
| 可视化管理 | ❌ | ✅ 宝塔面板 | ❌ |
| 生产环境推荐 | ❌ | ✅ 推荐 | ✅ 推荐 |
| 预计耗时 | 15-25分钟 | 30-50分钟 | 30-50分钟 |
| 外网可访问 | ❌ 仅本地 | ✅ | ✅ |

## 🚀 快速开始

选择您的部署方式，点击查看详细教程：

- **[🖥️ 本地电脑部署 →](/docs/deploy/local)** — 适合在 Windows/macOS 上快速体验
