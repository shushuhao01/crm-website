## 私有部署环境要求

### 服务器配置

| 配置项 | 最低要求 | 推荐配置 |
|--------|---------|---------|
| CPU | 2核 | 4核及以上 |
| 内存 | 4GB | 8GB及以上 |
| 硬盘 | 50GB | 100GB SSD |
| 带宽 | 5Mbps | 10Mbps及以上 |

### 操作系统

- **Ubuntu 24.04 LTS**（强烈推荐）
- Ubuntu 22.04 / 20.04 LTS
- Debian 11 / 12
- CentOS 7.x / 8.x
- Windows Server 2016+（不推荐）

### 软件环境

| 软件 | 版本要求 | 说明 |
|------|---------|------|
| Node.js | 18.x 或 20.x | JavaScript运行时 |
| MySQL | 5.7+ 或 8.0+ | 关系型数据库 |
| Nginx | 1.18+ | Web服务器/反向代理 |
| PM2 | 最新版 | Node.js进程管理器 |
| Git | 最新版 | 版本控制（可选） |

### 端口要求

| 端口 | 用途 |
|------|------|
| 80 | HTTP访问 |
| 443 | HTTPS访问 |
| 3000 | 后端API服务 |
| 3306 | MySQL数据库 |

### 网络要求

- 服务器需要能访问外网（用于物流查询API、短信服务等）
- 建议配置域名和SSL证书
- 如需APP使用，需要公网IP或域名
