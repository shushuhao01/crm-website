## 团队排名

在"业绩统计 → 团队业绩"中查看团队成员的业绩排名，激发团队竞争意识。

### 排名维度

- **销售额排名** — 按订单总金额排序
- **订单数排名** — 按已完成订单数排序
- **新客户数排名** — 按新增客户数量排序

### 排名周期

- 日排名：查看当天的实时排名
- 周排名：查看本周的累计排名
- 月排名：查看本月的累计排名
- 年度排名：查看年度累计排名

### 数据权限

| 角色 | 可查看范围 |
|------|-----------|
| 超级管理员/管理员 | 全公司所有部门和成员排名 |
| 部门经理 | 本部门所有成员排名 |
| 销售员 | 可查看排名列表，不含他人具体数据 |

### 业绩分享

部门经理及以上角色可以将业绩报表进行分享：
- 生成业绩报表图片
- 推送到企业微信群
- 推送到钉钉群
- 支持定时自动推送
