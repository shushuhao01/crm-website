## 数据导出

系统各模块均支持将数据导出为 Excel 文件，方便进行线下分析和存档。

### 支持导出的数据

| 模块 | 导出位置 | 导出内容 |
|------|---------|---------|
| 客户管理 | 客户列表页 → "导出"按钮 | 客户信息、分组、标签、跟进记录 |
| 订单管理 | 订单列表页 → "导出"按钮 | 订单详情、状态、金额、物流信息 |
| 业绩统计 | 各业绩页面 → "导出"按钮 | 业绩数据、排名统计 |
| 物流管理 | 物流列表页 → "导出"按钮 | 物流信息、快递状态 |
| 售后管理 | 售后列表页 → "导出"按钮 | 售后工单、处理记录 |

### 导出步骤

1. 进入对应的数据列表页面
2. 设置筛选条件缩小导出范围（可选）
3. 点击页面上方的"导出"按钮
4. 系统生成 Excel 文件并自动下载

### 注意事项

- 导出数据量较大时，系统可能需要几秒钟处理
- 导出的数据范围受当前用户的数据权限限制
- 建议先筛选再导出，避免导出不必要的数据
