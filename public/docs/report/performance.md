## 业绩统计

业绩统计模块帮助您实时查看个人和团队的销售业绩数据。

### 子模块

| 子模块 | 路径 | 权限 |
|--------|------|------|
| 个人业绩 | 业绩统计 → 个人业绩 | 所有角色 |
| 团队业绩 | 业绩统计 → 团队业绩 | 所有角色 |
| 业绩分析 | 业绩统计 → 业绩分析 | 部门经理及以上 |
| 业绩分享 | 业绩统计 → 业绩分享 | 部门经理及以上 |

### 个人业绩

查看您自己的销售业绩数据，包括：

- **订单数量**：已完成订单数
- **销售金额**：订单总金额
- **客户数量**：新增客户数
- **通话时长**：总通话时长（需开通电销功能）
- **转化率**：订单/客户转化率

支持按以下时间维度查看：
- 今日业绩
- 本周业绩
- 本月业绩
- 自定义时间范围

### 团队业绩

查看团队/部门的业绩汇总和成员排名：

**数据范围**：
- **部门经理**：查看本部门所有成员业绩
- **管理员**：查看全公司所有部门业绩
- **销售员**：查看团队排名（不含他人具体数据）

**排名展示**：
- 按销售额排名
- 按订单数排名
- 按新增客户数排名
- 支持日/周/月/年度排名

### 业绩分析

> 💡 仅部门经理及以上角色可见

提供多维度的业绩分析报表：
- **趋势分析**：业绩变化趋势图
- **同比分析**：与去年同期对比
- **环比分析**：与上月/上周对比
- **目标完成率**：业绩目标完成情况
