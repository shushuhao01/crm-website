## 发货管理

订单审核通过后进入"待发货"状态，发货操作在 **物流管理** 模块中进行。

### 单个发货

1. 进入"物流管理" → "发货列表"
2. 找到待发货的订单
3. 点击"发货"按钮
4. 选择快递公司
5. 填写快递单号
6. 确认发货

### 批量发货

1. 进入"物流管理" → "发货列表"
2. 勾选多个待发货订单
3. 点击"批量发货"
4. 下载发货模板Excel
5. 在模板中填写每个订单的快递公司和快递单号
6. 上传模板文件完成批量发货

### 物流跟踪

发货后可以实时跟踪物流状态：

1. 进入"物流管理" → "物流跟踪"
2. 查看各订单的物流轨迹
3. 系统支持对接多家主流快递公司，自动获取物流信息
4. 异常件（如滞留、退回）会自动预警提醒

### 物流状态更新

管理员可在"物流管理" → "状态更新"页面手动更新物流状态，例如标记为已签收、拒收、退回等。
