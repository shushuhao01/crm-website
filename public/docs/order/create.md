## 创建新订单

### 操作步骤

1. 进入"订单管理" → "新增订单"
2. **选择客户**：输入客户姓名或手机号搜索，选择已有客户；也可点击"新建客户"创建新客户
3. **选择商品**：从商品列表中选择商品，设置数量和单价（金额可手动调整）
4. **填写收货信息**：确认收货人姓名、电话、地址（可从客户信息自动填充）
5. **设置订单信息**：
   - 订单金额：系统自动计算，支持手动调整
   - 定金金额：客户已支付的定金
   - 支付方式：微信、支付宝、银行转账等
   - 订单备注：其他需要说明的信息
6. 点击"提交订单"

### 订单标记类型

| 标记 | 说明 |
|------|------|
| 正常单 | 正常发货的订单，会自动流转到审核 |
| 预留单 | 暂不发货的订单，需要手动提审 |
| 退单 | 需要退货处理的订单，需要手动提审 |

### 订单状态流转

订单从创建到完成经历以下状态：

```
待流转 → 待审核 → 待发货 → 已发货 → 已签收
```

**各状态说明：**

| 状态 | 说明 |
|------|------|
| 待流转 | 订单创建后的初始状态，等待自动流转或手动提审 |
| 待审核 | 订单已提交审核，等待管理员或客服审核 |
| 待发货 | 审核通过，等待物流发货 |
| 已发货 | 已填写物流单号，商品已发出 |
| 已签收 | 客户已签收商品，订单完成 |
| 审核拒绝 | 订单审核未通过，可修改后重新提审 |
| 已取消 | 订单已取消 |
| 拒收 | 客户拒收商品 |
| 物流退回 | 物流异常退回 |

### 订单流转配置

管理员可在"系统管理" → "系统设置"中配置订单流转模式：
- **自动流转**：正常单提交后自动进入待审核状态
- **手动提审**：所有订单需要手动点击提审
