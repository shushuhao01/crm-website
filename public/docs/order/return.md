## 售后管理

云客CRM拥有独立的 **售后管理** 模块，处理客户的退换货、投诉、补发等售后事务。

### 售后类型

| 类型 | 说明 |
|------|------|
| 退货退款 | 客户退回商品并退款 |
| 换货 | 更换商品 |
| 仅退款 | 不退货只退款 |
| 补发 | 补发商品 |
| 投诉 | 客户投诉处理 |
| 咨询 | 售后咨询 |

### 创建售后工单

1. 进入"售后管理" → "新建售后"
2. 填写必填信息：
   - **关联订单号**：选择需要售后的原订单
   - **售后类型**：选择退货退款/换货/补发等
   - **售后原因**：说明售后原因
   - **处理方式**：选择处理方式
3. 填写选填信息（如适用）：
   - 退款金额
   - 退货物流单号
   - 图片凭证
   - 备注说明
4. 提交工单

### 售后流程

1. **创建工单** — 填写售后信息，关联原订单
2. **审核** — 客服或管理员审核售后申请
3. **处理** — 执行退款、换货、补发等操作
4. **完成** — 售后处理完成，关闭工单

### 售后数据

进入"售后管理" → "售后数据"可查看：
- 售后工单数量统计
- 各售后类型占比
- 处理时效分析
- 退款金额统计
- 售后率分析

> 💡 售后管理权限：管理员、客服可查看所有售后工单；销售员可查看自己客户的售后工单。
