## 快速开始使用云客CRM

按照以下步骤，5分钟内即可开始使用系统：

### 第1步：了解数据看板（30秒）

登录系统后，首页即为 **数据看板**，展示：
- 今日/本周/本月核心业务数据
- 订单数量、销售金额等关键指标
- 待处理事项提醒

### 第2步：添加员工账号（1分钟）

1. 进入"系统管理" → "用户管理"
2. 点击"添加用户"
3. 填写员工姓名、手机号，设置密码
4. 选择所属部门，分配角色（如：销售员）

### 第3步：录入客户数据（2分钟）

**方式一：手动添加**
1. 进入"客户管理" → "新增客户"
2. 填写客户姓名和手机号（必填）
3. 补充微信号、分组、标签等信息（可选）
4. 点击"保存"

**方式二：批量导入**
1. 进入"客户管理" → "客户列表"
2. 点击"导入客户"按钮
3. 下载Excel导入模板
4. 按模板格式填写客户信息，上传文件完成导入

### 第4步：创建第一个订单（1分钟）

1. 进入"订单管理" → "新增订单"
2. 搜索并选择客户（或新建客户）
3. 选择商品，设置数量和价格
4. 确认收货地址
5. 点击"提交订单"

### 第5步：配置商品信息（30秒）

1. 进入"商品管理" → "商品列表"
2. 点击"新增商品"
3. 填写商品名称、价格、库存等信息
4. 保存后即可在创建订单时选择

> 💡 提示：建议按照"设置 → 添加员工 → 录入客户 → 添加商品 → 创建订单"的顺序来初始化系统，效果最佳。
