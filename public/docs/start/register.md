## 注册账号

云客CRM提供 **7天免费试用**，注册流程如下：

1. 访问云客CRM官网，点击"免费试用 7 天"按钮
2. 填写注册信息：
   - **企业名称**（必填）：您的公司或团队名称
   - **联系人姓名**（必填）：管理员姓名
   - **手机号码**（必填）：用于登录和接收验证码
   - **登录密码**（必填）：设置您的登录密码
3. 获取并输入手机短信验证码
4. 点击"注册"完成账号创建

注册成功后，系统会自动为您创建企业空间，您将成为该企业的 **超级管理员**，拥有全部权限。

> 💡 免费试用期为 7 天，试用期内可体验全部功能。试用到期后需要购买套餐以继续使用。

## 登录系统

1. 访问系统登录页面
2. 输入注册时使用的手机号和密码
3. 点击"登录"按钮进入系统
4. 首次登录会进入数据看板页面

### 忘记密码

如果忘记登录密码，请联系您的企业管理员（超级管理员或管理员），由管理员在"系统管理 → 用户管理"中为您重置密码。

### 多设备登录

系统支持同时在PC端浏览器和移动APP上登录使用，数据实时同步。
