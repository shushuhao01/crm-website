## 什么是云客CRM？

云客CRM是一款专为销售团队打造的智能客户关系管理系统，集客户管理、订单处理、业绩分析、物流跟踪、售后管理、财务管理于一体，助力企业实现销售业绩持续增长。

系统支持 **SaaS云端版** 和 **私有部署版** 两种模式，同时提供 **移动APP** 和 **PC端** 双端使用。

### 核心功能模块

| 模块 | 功能说明 |
|------|---------|
| 📊 数据看板 | 实时业务数据概览，关键指标一目了然 |
| 👥 客户管理 | 客户列表、新增客户、客户分组、客户标签 |
| 📦 订单管理 | 订单列表、新增订单、订单审核、代收管理 |
| 📈 业绩统计 | 个人业绩、团队业绩、业绩分析、业绩分享 |
| 🚚 物流管理 | 物流列表、发货管理、物流跟踪、状态更新 |
| 🔧 售后管理 | 售后订单、新建售后、售后数据统计 |
| 📁 资料管理 | 资料列表、客户查询、回收站 |
| 💰 财务管理 | 绩效数据、代收管理、增值管理、结算报表 |
| 🛍️ 商品管理 | 商品列表、商品分类、库存管理、商品分析 |
| 📞 服务管理 | 短信管理、通话管理、录音管理 |
| ⚙️ 系统管理 | 用户管理、角色权限、部门管理、系统设置 |

### 产品优势

- **开箱即用** — 注册即可使用，5分钟快速上手
- **数据安全** — 银行级加密保护，多重备份机制
- **灵活部署** — 支持SaaS云端和私有部署两种模式
- **移动办公** — 配套移动APP，支持外呼、客户管理
- **角色权限** — 精细化权限控制，支持超级管理员、管理员、部门经理、销售员、客服五种角色
- **持续更新** — 产品团队持续迭代，紧跟行业趋势
