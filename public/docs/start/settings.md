## 企业信息设置

登录系统后，建议先完善企业基础信息：

1. 进入左侧菜单"系统管理" → "系统设置"
2. 在"基础设置"标签页中填写：
   - 企业名称和Logo
   - 系统名称
   - 联系方式
3. 点击"保存"完成设置

## 部门架构设置

根据企业组织架构创建部门：

1. 进入"系统管理" → "部门管理"
2. 点击"添加部门"创建一级部门（如：销售部）
3. 可以继续添加子部门（如：销售一组、销售二组）
4. 设置部门负责人
5. 支持多级部门结构，例如：公司 → 销售部 → 销售一组

## 角色权限配置

系统预置了五种角色，也支持自定义角色：

| 角色 | 说明 | 数据范围 |
|------|------|---------|
| 超级管理员 | 拥有所有权限，包含系统配置 | 全部数据 |
| 管理员 | 日常业务管理和用户管理 | 全部数据 |
| 部门经理 | 管理本部门的业务和团队 | 本部门数据 |
| 销售员 | 基础销售功能，客户和订单管理 | 个人数据 |
| 客服 | 客户服务、订单审核等 | 全部数据 |

进入"系统管理" → "角色权限"可查看和配置各角色的详细权限。

## 添加员工账号

1. 进入"系统管理" → "用户管理"
2. 点击"添加用户"
3. 填写员工信息（姓名、手机号、密码）
4. 选择所属部门
5. 分配角色
6. 点击"保存"完成创建
