## 部门管理

管理企业的组织架构，支持多级部门结构。

> ⚠️ 仅管理员和超级管理员可操作

### 创建部门

1. 进入"系统管理" → "部门管理"
2. 点击"新增部门"
3. 填写部门信息：
   - **部门名称**（必填）
   - **上级部门**（可选，不选则为顶级部门）
   - **部门负责人**（可选）
   - **部门描述**（可选）
4. 点击"确定"保存

### 部门层级

支持多级部门结构，例如：

```
公司（顶级）
├── 销售部
│   ├── 销售一组
│   └── 销售二组
├── 客服部
└── 运营部
```

### 部门操作

| 操作     | 说明                     |
|----------|--------------------------|
| 编辑     | 修改部门名称、负责人等信息 |
| 删除     | 删除部门（部门下有成员时不可删除） |
| 查看成员 | 查看部门下的所有成员列表     |
| 添加子部门 | 在当前部门下创建子部门       |

### 部门与数据权限

部门结构直接影响数据权限范围：
- 部门经理可查看本部门及其子部门的所有数据
- 销售员仅可查看自己的数据
- 管理员可查看所有部门的数据
