## 系统配置

在"系统管理" → "系统设置"中可以配置系统的各项参数。

### 配置标签页

系统设置包含以下配置模块（根据管理员权限和系统开通的功能模块，实际可见的标签页可能不同）：

| 标签页 | 说明 |
|--------|------|
| 安全配置 | 登录安全策略、密码规则、会话超时等 |
| 通话配置 | 电销外呼参数、线路配置（需开通电销功能） |
| 邮件配置 | 邮件服务器SMTP配置 |
| 短信配置 | 短信服务商配置（阿里云短信等） |
| 存储配置 | 文件存储方式，支持本地存储和云存储（阿里云OSS） |
| 商品配置 | 商品相关的默认设置 |
| 系统监控 | 服务器资源使用情况监控（超管可见） |
| 订单配置 | 订单流转模式、自动流转延迟时间等 |
| 数据备份 | 数据备份设置和备份记录 |
| 通知配置 | 消息通知渠道和规则配置 |
| 用户协议 | 管理用户协议和隐私政策内容 |
| 移动应用 | 移动APP相关配置和SDK管理 |
| 系统日志 | 查看操作日志和系统运行日志 |

### 常用配置

**订单配置**：
- 流转模式：自动流转 / 手动提审
- 流转延迟：自动流转的等待时间（分钟）

**安全配置**：
- 登录失败锁定策略
- 密码强度要求
- 会话超时时间

**存储配置**：
- 本地存储：文件保存在服务器本地
- 阿里云OSS：文件上传到云端存储

> 💡 部分配置可能由管理后台（Admin）统一管控，此时会显示"该配置由管理后台统一管控"提示。
