## 角色权限管理

通过角色来管理用户的功能权限和数据访问范围。

### 系统预置角色

| 角色 | 说明 | 数据范围 | 可编辑权限 |
|------|------|---------|-----------|
| 超级管理员 | 系统最高权限，拥有全部功能 | 全部数据 | 否（固定权限） |
| 管理员 | 日常业务管理和用户管理 | 全部数据 | 是 |
| 部门经理 | 管理本部门的业务和团队 | 本部门数据 | 是 |
| 销售员 | 基础销售功能 | 个人数据 | 是 |
| 客服 | 订单审核、售后处理等 | 全部数据 | 是 |

### 自定义角色

除系统预设角色外，可以创建自定义角色：

1. 进入"系统管理" → "角色权限"
2. 点击"新增角色"
3. 输入角色名称和描述
4. 在权限树中勾选需要的权限
5. 点击"保存"

### 权限配置

1. 选择要配置权限的角色
2. 点击"配置权限"按钮
3. 在权限树中勾选/取消勾选权限项
4. 权限分类包括：
   - 客户管理权限
   - 订单管理权限
   - 业绩统计权限
   - 物流管理权限
   - 售后管理权限
   - 财务管理权限
   - 系统管理权限
5. 点击"保存"应用配置

> 💡 系统预置的"超级管理员"角色权限不可修改，始终拥有全部权限。
