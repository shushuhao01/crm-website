## 用户管理

管理系统中的所有用户账号，包括添加、编辑、禁用、删除等操作。

> ⚠️ 仅管理员和超级管理员可操作

### 添加用户

1. 进入"系统管理" → "用户管理"
2. 点击"添加用户"
3. 填写用户信息：
   - **用户名**（必填）：用于登录
   - **真实姓名**（必填）：用户显示名称
   - **手机号**（必填）：联系电话
   - **所属部门**（必填）：选择部门
   - **角色**（必填）：选择角色（超级管理员、管理员、部门经理、销售员、客服）
   - **初始密码**（必填）：设置登录密码
   - **邮箱**（可选）
4. 点击"确定"保存

### 用户操作

| 操作 | 说明 |
|------|------|
| 编辑 | 修改用户基本信息 |
| 重置密码 | 重置用户的登录密码 |
| 禁用/启用 | 暂停或恢复用户使用权限，保留数据 |
| 删除 | 永久移除用户账号 |
| 分配角色 | 修改用户的角色 |
| 调整部门 | 修改用户所属部门 |

### 批量操作

- 批量导入用户
- 批量调整部门
- 批量禁用/启用
