# 获客助手与活码

## 获客链接

### 创建获客链接
1. 进入「获客助手」模块
2. 点击「创建获客链接」
3. 设置链接名称和分配规则
4. 配置欢迎语和自动标签
5. 生成链接或二维码

### 智能分配
- 轮询分配：按顺序轮流分配给员工
- 加权分配：按权重比例分配
- 在线优先：优先分配给在线员工

### 数据分析
- 访问量/添加量/留存率
- 转化漏斗分析
- 成员排名对比
- 时间段趋势图

## 渠道活码

### 活码管理
- 支持创建多个渠道活码
- 每个活码绑定多个员工
- 扫码后自动分配对应员工

### 活码统计
- 每日扫码数据统计
- 渠道对比分析
- 员工接待数据

## 智能规则

### 获客智能规则
- 基于时间段的分配策略
- 基于客户标签的精准分配
- 自动打标签与分组
