# AI助手与知识库

## AI智能助手

### 功能概述
AI助手集成大语言模型能力，为客服人员提供智能回复建议、自动质检和知识库问答。

### AI模型管理
- 支持接入多种AI模型（如通义千问、文心一言等）
- 配置API密钥和调用参数
- 设置调用额度和费率

### 智能体管理
- 创建自定义AI智能体
- 设置系统提示词和行为规则
- 关联知识库数据源
- 调整回复风格和专业度

### AI回复建议
- 客服对话时实时生成回复建议
- 一键采纳或编辑后发送
- 回复质量持续优化

## 知识库管理

### 创建知识库
1. 进入「知识库管理」
2. 创建知识库并设置分类
3. 添加知识条目（FAQ、文档等）
4. 关联到AI智能体

### 知识条目
- 支持问答对格式
- 支持长文档格式
- 支持标签分类
- 支持搜索和筛选

## AI质检

### 质检策略
- 配置AI质检评分维度
- 设置质检标准和权重
- 选择质检范围和频率

### 质检结果
- 自动生成质检评分
- 标记问题会话
- 改进建议报告
- 趋势分析图表

## 话术库

### 话术管理
- 按分类组织常用话术
- 支持变量替换
- 侧边栏快捷引用
- 团队共享话术库
