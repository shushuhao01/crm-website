# 企微配置与授权

## 授权模式

系统支持两种企微授权模式：

### 自建应用模式
适用于：企业自行开发管理

1. 登录企业微信管理后台
2. 创建自建应用，获取 CorpID 和 Secret
3. 在CRM「企微配置」页面填入凭证
4. 配置回调URL并验证

### 服务商模式（推荐）
适用于：SaaS多租户部署

1. 管理后台配置服务商Suite应用信息
2. 租户在CRM中点击「授权接入」
3. 跳转企微授权页面完成一键授权
4. 系统自动获取Token并同步数据

## 配置步骤

### 第一步：基础信息
- 填入企业CorpID
- 配置应用AgentID和Secret
- 设置可信域名

### 第二步：回调配置
- 配置回调URL（系统自动生成）
- 填入Token和EncodingAESKey
- 点击「测试连接」验证

### 第三步：权限申请
- 通讯录读取权限
- 客户联系权限
- 会话存档权限（需额外购买）

### 第四步：功能启用
- 选择需要启用的功能模块
- 配置数据同步策略
- 测试各功能是否正常

## 常见问题

**Q: 授权后无法同步通讯录？**
A: 请检查应用的可见范围是否包含了需要同步的部门。

**Q: 回调验证失败？**
A: 确认回调URL可从外网访问，Token和AESKey与企微后台一致。
