# 对外收款管理

## 收款码管理

### 创建收款码
1. 进入「企微收款」模块
2. 点击「创建收款码」
3. 设置金额类型（固定/自定义）
4. 分配使用成员
5. 生成二维码

### 收款码类型
- **固定金额码**：客户扫码后自动显示预设金额
- **自定义金额码**：客户扫码后自行输入金额

## 收款记录

### 记录查看
- 按时间范围筛选
- 按成员筛选
- 按金额范围筛选
- 导出收款明细

### 数据统计
- 总收款金额和笔数
- 成员收款排名
- 日/周/月趋势图

## 退款管理

### 发起退款
1. 找到原收款记录
2. 点击「退款」按钮
3. 填写退款金额和原因
4. 确认提交

### 退款状态
- 处理中：退款申请已提交
- 已完成：退款成功到账
- 已拒绝：退款申请被驳回
