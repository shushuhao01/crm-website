# 会话存档与质检

## 会话存档

### 功能概述
会话存档功能可以保存员工与客户之间的聊天记录，用于合规审查、服务质量评估和争议处理。

### 前提条件
- 企业已在企业微信后台开通会话存档功能
- 已购买会话存档许可（按人数计费）
- 在CRM中配置存档密钥

### 使用方式
1. 在「会话存档」模块查看存档记录
2. 支持按员工、时间、关键词搜索
3. 查看完整对话内容
4. 导出记录用于合规审查

## 敏感词监控

### 设置敏感词
1. 进入「敏感词管理」
2. 添加敏感词或词组
3. 设置监控范围和告警方式

### 告警处理
- 命中敏感词后自动生成告警记录
- 管理员收到实时通知
- 查看上下文对话内容
- 标记处理结果

## 质检管理

### 手动质检
- 选择需要质检的会话
- 按质检规则逐项评分
- 生成质检报告

### AI智能质检
- 配置AI质检策略
- 系统自动分析会话内容
- 自动评分和生成分析报告
- 发现问题自动预警

## 防骚扰规则

### 规则配置
- 关键词拦截：阻止发送含特定词的消息
- 链接拦截：控制外部链接发送
- 频率限制：限制单位时间内消息数量
- 豁免设置：管理员和特定角色可豁免
