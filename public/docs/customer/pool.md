## 客户分组与标签

通过客户分组和标签来分类管理客户，便于筛选和统计。

> ⚠️ 客户分组和标签的创建和管理仅管理员可操作

### 客户分组

客户分组用于对客户进行分类管理。

**创建分组：**
1. 进入"客户管理" → "客户分组"
2. 点击"新增分组"按钮
3. 输入分组名称和描述
4. 点击"确定"保存

**分组管理：**
- 编辑：修改分组名称和描述
- 删除：删除分组（分组下有客户时不可删除）
- 排序：拖拽调整分组显示顺序

**使用分组：**
- 新增客户时选择所属分组
- 在客户列表中按分组筛选客户
- 编辑客户时可修改所属分组

### 客户标签

客户标签用于标记客户特征，构建客户画像，支持多选。

**创建标签：**
1. 进入"客户管理" → "客户标签"
2. 点击"新增标签"按钮
3. 输入标签名称
4. 点击"确定"保存

**使用标签：**
- 新增客户时选择标签（可多选）
- 在客户列表中按标签筛选客户
- 编辑客户时可修改标签

> 💡 合理使用分组和标签可以帮助您更高效地管理和筛选客户资源。
