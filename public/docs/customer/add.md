## 添加新客户

### 手动添加

1. 进入左侧菜单"客户管理" → "新增客户"
2. 填写客户基本信息
3. 点击"保存"完成添加

### 客户字段说明

**必填字段：**

| 字段 | 说明 |
|------|------|
| 客户姓名 | 客户的真实姓名 |
| 手机号码 | 客户的联系电话 |

**选填字段：**

| 字段 | 说明 |
|------|------|
| 微信号 | 客户的微信号 |
| 性别 | 男 / 女 / 未知 |
| 年龄 | 客户年龄 |
| 客户分组 | 选择所属分组（由管理员预设） |
| 客户标签 | 选择标签，可多选（由管理员预设） |
| 收货地址 | 省、市、区、详细地址 |
| 备注 | 其他补充信息 |

### 批量导入

1. 进入"客户管理" → "客户列表"
2. 点击"导入客户"按钮
3. 下载Excel导入模板
4. 按模板格式填写客户信息（姓名和手机号必填）
5. 上传文件，系统自动导入

### 数据权限说明

- **销售员**：只能看到自己名下的客户
- **部门经理**：可以查看本部门所有客户
- **管理员/超级管理员**：可以查看全部客户
