# 自动发送规则

## 概述

自动发送规则可以在特定事件触发时自动向客户发送短信，无需人工操作。

## 创建规则

### 步骤
1. 进入「短信管理」-「自动发送」
2. 点击「创建规则」
3. 选择触发事件
4. 选择短信模板
5. 配置发送条件
6. 启用规则

## 支持的触发事件

| 事件 | 说明 |
|------|------|
| 订单确认 | 订单审核通过时 |
| 订单发货 | 物流发出时 |
| 订单签收 | 快递签收时 |
| 客户生日 | 客户生日当天 |
| 跟进提醒 | 跟进时间到期时 |
| 付款成功 | 客户完成支付时 |

## 发送条件

### 时间限制
- 设置可发送时段（如9:00-21:00）
- 避免在休息时间打扰客户

### 频率限制
- 同一客户每日最多发送N条
- 避免过度骚扰

### 角色权限
- 指定哪些角色的操作会触发发送
- 指定发送人归属部门

## 规则管理

- 启用/禁用规则
- 查看规则执行日志
- 修改规则配置
- 删除不需要的规则
