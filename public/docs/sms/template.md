# 模板管理与审核

## 创建模板

### 步骤
1. 进入「短信管理」-「模板管理」
2. 点击「申请模板」
3. 选择模板类型（通知/营销/验证）
4. 填写模板内容（使用变量占位符）
5. 提交审核

### 变量使用
模板中可使用以下变量：
- `${客户姓名}` - 客户姓名
- `${订单号}` - 订单编号
- `${商品名称}` - 商品名称
- `${金额}` - 订单金额
- 更多变量请查看「变量文档」

## 四阶段审核流程

```
提交申请 → 管理员审核 → 服务商审核 → 激活可用
```

### 第一阶段：提交
用户创建模板并提交审核申请。

### 第二阶段：管理员审核
管理后台管理员审核模板内容：
- 通过（有服务商模板CODE）→ 直接激活
- 通过（无CODE）→ 进入服务商审核
- 拒绝 → 退回并附拒绝原因

### 第三阶段：服务商审核
管理员将模板提交至短信服务商平台审核。

### 第四阶段：激活
管理员填入服务商返回的模板CODE，模板激活可用。

## 预设模板
系统提供多个预设通用模板，可直接使用无需审核。
