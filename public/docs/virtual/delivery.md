# 自动发货与领取

## 自动发货流程

```
订单确认 → 匹配库存 → 分配卡密/资源 → 生成领取链接 → 发送通知邮件
```

### 发货触发条件
- 订单状态变为「已确认」
- 商品类型为虚拟商品
- 对应库存充足

### 发货记录
每次发货自动生成记录：
- 订单号和商品信息
- 分配的卡密或资源
- 操作人和时间
- 邮件发送状态

## 客户领取

### 领取方式
1. **链接领取**：客户点击邮件中的领取链接
2. **手动发送**：客服手动复制卡密给客户

### 领取流程
1. 客户访问领取链接
2. 验证身份（密码/短信）
3. 查看并复制卡密或资源链接
4. 确认领取完成

## 邮件通知

### 自动发送
开启后，发货完成自动发送通知邮件给客户。

### 邮件内容
- 订单信息
- 领取链接
- 使用说明
- 有效期提醒
