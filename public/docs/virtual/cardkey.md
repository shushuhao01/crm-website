# 卡密管理

## 卡密导入

### 单个添加
1. 进入「虚拟商品」-「卡密管理」
2. 选择关联商品
3. 输入卡密内容
4. 保存入库

### 批量导入
1. 下载导入模板
2. 填入卡密数据
3. 上传文件自动解析
4. 确认导入结果

## 卡密状态

| 状态 | 说明 |
|------|------|
| 未使用 | 库存中待分配 |
| 已预占 | 已关联订单待发货 |
| 已使用 | 已发货给客户 |
| 已领取 | 客户已确认领取 |
| 已过期 | 超过有效期 |
| 已作废 | 手动作废 |

## 库存统计
- 各状态卡密数量
- 按商品分组统计
- 库存消耗速率
- 低库存预警
