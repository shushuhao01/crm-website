# 领取页面配置

## 发货模式

### 链接模式（推荐）
- 自动生成领取链接
- 客户自助领取
- 减少人工操作

### 手动模式
- 客服手动发货
- 适合需要额外说明的商品

## 领取页面设置

### 链接有效期
设置领取链接的有效天数（默认30天），过期后链接失效。

### 登录方式
- **密码验证**：使用初始密码登录
- **短信验证**：发送验证码到客户手机

### 页面提示语
自定义领取页面顶部显示的提示信息。

## 邮箱配置

### 官方邮箱
使用系统默认的发件邮箱（无需额外配置）。

### 自定义邮箱
配置企业自己的SMTP邮箱：
1. 填写SMTP服务器地址和端口
2. 填写发件邮箱和密码
3. 点击「验证」测试连通性
4. 验证通过后保存

### 邮件模式
- **链接模式**：邮件中包含领取链接
- **内容模式**：邮件中直接包含卡密/资源
- **混合模式**：同时包含链接和内容摘要
